Momentum - Personal Productivity Tracker
A sleek, client-side productivity app to track your focused work sessions. Log your progress, manage categories, and stay motivated with this minimalist time tracker. Built with pure HTML, CSS, and JS.

✨ Features
User Accounts: Simple sign-up and login system using email and password. All data is stored locally and separated by user.

Dynamic Category Management: Add, edit, and delete your work categories on the fly.

Custom Colors: Choose a unique color for each category for better visual organization.

One-Click Logging: Add a 45-minute focused work unit with a single click.

Interactive Daily Log: Visualize your daily progress with colored blocks. Hover to see a delete icon and easily remove accidental entries.

Total Progress Tracking: A prominent "Total Momentum" counter keeps you motivated by showing your total logged units.

Data Portability: Export your data to a JSON file for backup or import it to a new device.

Responsive Design: Looks great on both desktop and mobile devices.

🚀 How to Use
Option 1: GitHub Pages (Recommended)
The easiest way to use this app is to deploy it on GitHub Pages.

Create a new public repository on GitHub.

Upload the index.html, style.css, and app.js files to your new repository.

In your repository settings, go to the "Pages" tab.

Under "Build and deployment", select Deploy from a branch, choose the main branch and / (root) folder, and click Save.

Your app will be live at https://<your-username>.github.io/<your-repo-name>/ in a few minutes.

Option 2: Local Usage
Download the index.html, style.css, and app.js files.

Place them all in the same folder on your computer.

Open the index.html file in your preferred web browser (like Chrome, Firefox, or Edge).

⚠️ Important Note on Security
This application uses localStorage and sessionStorage for all its data handling, meaning everything is stored on your computer in your web browser. The password for your account is also stored this way and is not encrypted.

This system is designed for personal convenience and is NOT secure for sensitive information or a public environment. It serves as a simple barrier to prevent others from casually accessing your log on a shared device.

🛠️ Tech Stack

CSS3 (with CSS Variables for easy theming)

No frameworks, no libraries, no backend. Just pure, simple web technologies.
